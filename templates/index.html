About yotweek

yotweek is a powerful, browser-based video downloader and editor powered by yt-dlp. It lets you download videos from YouTube and 1000+ other platforms, then convert, trim, or enhance them — all from one page.

Tech Stack
  • yt-dlpHigh-performance video extraction engine supporting 1000+ sites
  • FastAPI + Socket.IOFast, async backend with real-time progress updates via WebSockets
  • FFmpegProfessional-grade video and audio processing
Privacy

Files are stored temporarily and automatically deleted after a few minutes. No account required. No data is retained beyond your session.

Recommended Features
  • HD Video Download Download videos in Full HD, HD, or Standard quality from YouTube and 1000+ sites.
  • Playlist & Channel Downloads Paste a YouTube playlist or channel URL and download all videos in one go.
  • Batch URL Downloads Paste up to 20 URLs at once and start parallel downloads.
  • Video Conversion Convert any downloaded file to MP4, WebM, AVI, MP3, or WAV with custom bitrates.
  • Trim & Clip Extraction Keep only the portion you need — cut by start/end time or extract a short clip.
  • Export as ZIP Select multiple files and download them all as a single ZIP archive.

yotweek — Help Guide

Single Video Download
  1. Paste a video URL (YouTube, TikTok, Instagram, Twitter/X, Facebook and 1,000+ other sites) into the URL box.
  2. Click Get Info to fetch the video title, duration, and thumbnail.
  3. Choose your preferred Quality (e.g. HD 720p, Full HD 1080p) and File Format (mp4, webm, mkv, mp3…).
  4. Click Download — the file is prepared on the server and a download link appears automatically.
Playlist Download
  1. Switch to the Playlist / Batch tab, then select the Playlist sub-tab.
  2. Paste a YouTube playlist URL (or any playlist-capable URL).
  3. Set the maximum number of videos to download, plus quality and format.
  4. Click Download Playlist. Each video is queued and processed concurrently.
Batch Download
  1. Switch to the Playlist / Batch tab, then select the Batch URLs sub-tab.
  2. Paste any text containing video URLs — from one site or many. URLs are detected automatically and arranged one per line regardless of how they were copied.
  3. Up to 20 URLs are supported per batch. Extra URLs beyond 20 are discarded.
  4. Choose quality and format, then click Start Batch Download.
Export as ZIP
Download multiple already-processed files at once. Select files from your download list using the checkboxes, then click Download ZIP to receive them in a single archive.
Edit & Convert
  • Convert — change an existing file's format or resolution (e.g. mp4 → mp3, 1080p → 480p).
  • Trim — cut a video to a specific start/end time.
  • Merge Audio — overlay a separate audio file onto a video track.
  • Batch Convert — convert multiple files at once using the same settings.
File Manager
All your processed files are listed here. You can preview videos in the built-in player, download individual files, or delete files to free up server space.
Moving the Help Button
Drag the ? button to any corner of the screen. A short click (no drag) opens this guide; a longer press-and-drag repositions it.
